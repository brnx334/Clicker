## General Info
Android mobile game. Tap to earn points, strategically upgrade your clicking power, and compete globally for the top spot. Easy to play, suitable for all ages.



## Technologies
Project is created with:
* Kotlin
* Jatpack Compose
* Room database
* Toasty
* Coroutines
* Play Games SDK
* Google cloud
* Oauth2
